# Add Mode

Read first:

1. `AGENTS.md`
2. `docs/modes/shared/agent-modes.md`
3. `docs/modes/shared/agent-skills.md`
4. `docs/modes/add/add-mode-guide.md`
5. `docs/modes/add/requirement-routing-rules.md`
6. the current task plan and worklog
7. the closest existing skill/docs files for the requested behavior

Focus on:

- durable user requirements
- repo-owned versus host-owned boundaries
- choosing the right instruction layer
- additive customization of docs, skills, metadata, templates, or checks
- keeping repeated chat guidance discoverable in later sessions

Prefer the smallest set of edits that makes the new behavior reusable.
